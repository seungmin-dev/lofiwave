### 🖥 페이지 설명

유튜브 검색 api로 lofi 음악을 50개 리스트로 받아옵니다. 목록에서 원하는 음악을 선택해서 들을 수 있고, 엘피바 하단의 조작버튼으로 앞,뒤 음악 이동이 가능하고 멈춤,재생이 가능합니다.<br/>
해당 음악영상의 이미지를 배경과 엘피판에 적용해 조금 더 몰입감 있는 환경을 조성했습니다.
